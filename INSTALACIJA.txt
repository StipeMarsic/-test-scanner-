TEST SCANNER - UPUTE ZA INSTALACIJU
====================================

PREDUVJETI:
----------
1. Flutter SDK (3.0.0 ili noviji)
2. Android Studio ili VS Code
3. Android SDK (za Android uređaje)
4. Xcode (za iOS uređaje - samo na macOS)
5. Fizički uređaj ili emulator

KORAK 1: INSTALACIJA FLUTTER SDK
-------------------------------
1. Preuzmite Flutter SDK s: https://flutter.dev/docs/get-started/install
2. Raspakujte u željenu mapu (npr. C:\flutter)
3. Dodajte Flutter u PATH varijablu
4. Otvorite Command Prompt i pokrenite: flutter doctor
5. Riješite sve probleme koje Flutter doctor prijavi

KORAK 2: INSTALACIJA ANDROID STUDIO
----------------------------------
1. Preuzmite Android Studio s: https://developer.android.com/studio
2. Instalirajte Android Studio
3. Otvorite Android Studio i instalirajte Android SDK
4. Kreirajte Android emulator (opcijalno)

KORAK 3: PROVJERA INSTALACIJE
----------------------------
Pokrenite sljedeće naredbe u Command Prompt:

flutter doctor -v

Provjerite da su svi dijelovi označeni zeleno (✓).

KORAK 4: KLONIRANJE PROJEKTA
---------------------------
1. Otvorite Command Prompt
2. Navigirajte do željene mape
3. Pokrenite: git clone <repository-url>
4. Navigirajte u mapu projekta: cd TestScanner

KORAK 5: INSTALACIJA DEPENDENCIES
--------------------------------
Pokrenite sljedeće naredbe:

flutter pub get

KORAK 6: PROVJERA UREĐAJA
-------------------------
Pokrenite:

flutter devices

Provjerite da je vaš uređaj ili emulator prikazan.

KORAK 7: POKRETANJE APLIKACIJE
-----------------------------
Pokrenite:

flutter run

ILI

Pokrenite run_app.bat datoteku (Windows).

KONFIGURACIJA ZA ANDROID:
------------------------
1. Otvorite android/app/src/main/AndroidManifest.xml
2. Provjerite da su sve potrebne dozvole dodane:
   - CAMERA
   - WRITE_EXTERNAL_STORAGE
   - READ_EXTERNAL_STORAGE
   - INTERNET

KONFIGURACIJA ZA iOS:
--------------------
1. Otvorite ios/Runner/Info.plist
2. Dodajte potrebne dozvole:
   - NSCameraUsageDescription
   - NSPhotoLibraryUsageDescription

RJEŠAVANJE ČESTIH PROBLEMA:
--------------------------

PROBLEM: Flutter nije prepoznat
RJEŠENJE: Dodajte Flutter u PATH varijablu

PROBLEM: Android SDK nije pronađen
RJEŠENJE: Instalirajte Android Studio i Android SDK

PROBLEM: Kamera ne radi
RJEŠENJE: Provjerite dozvole u AndroidManifest.xml

PROBLEM: Aplikacija se ruši
RJEŠENJE: 
1. flutter clean
2. flutter pub get
3. flutter run

PROBLEM: Dependencies se ne mogu instalirati
RJEŠENJE:
1. Provjerite internet vezu
2. Pokušajte s drugim pub serverom
3. Provjerite Flutter verziju

PROBLEM: Emulator se ne pokreće
RJEŠENJE:
1. Provjerite da je Intel HAXM instaliran
2. Provjerite da je virtualization omogućen u BIOS-u
3. Pokušajte s drugim emulatorom

KORISNE NAREDBE:
---------------
flutter doctor -v          # Detaljna provjera instalacije
flutter clean             # Očišćavanje cache-a
flutter pub get           # Instalacija dependencies
flutter devices           # Prikaz dostupnih uređaja
flutter run               # Pokretanje aplikacije
flutter build apk         # Kreiranje APK datoteke
flutter build ios         # Kreiranje iOS aplikacije
